# E-commerce Impressão 3D

Sistema Completo de vendas online , com produtos feitos em Impressora 3D.

## Funcionalidades 

- Gerenciamento de produtos(imagens,valor,nome,descrição,etc)
- Montagem das páginas de forma dinâmica apartir das requisições no banco de dados
- Requisição do Serviço dos Correios para Calculo de frete 
- SASS
- Design Responsivo 
## Tecnologias utilizadas



- Interface construida apartir da biblioteca **javascript React** 
- Banco de Dados relacional **Mysql**
- Requisição do Banco de dados utilizando **PHP**
